# GitHub Copilot

> GitHub's AI coding assistant. Plugin for VS Code, JetBrains, and others.

---

## What It Is
Inline code suggestions that appear as you type, plus a chat panel for longer interactions. Integrates with GitHub ecosystem (PRs, issues, actions).

## Key Features

### AGENTS.md
Project-level instructions file (similar to CLAUDE.md). Tells Copilot about your conventions and stack.

### Copilot Chat
Chat panel in the IDE for longer questions and multi-step tasks.

### Copilot Workspace
Task-oriented workflow: start from a GitHub issue, Copilot proposes a plan and code changes.

## Best For
- Quick inline completions (tab-complete style)
- Teams already on GitHub with existing CI/CD
- Minimal disruption to existing IDE setup
- Boilerplate and repetitive patterns

## Limitations
- Agent autonomy less developed than Claude Code
- Limited hook/verification system
- Inline suggestions can be distracting

## Harness Ceiling
Level 2. AGENTS.md provides context, but limited enforcement.

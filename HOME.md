## Core Principles
1. **Foundations over frameworks** — learn the pattern, not just the tool
2. **Enforce, don't document** — if it matters, make it a linter rule, not a README line
3. **Audit-ready always** — build like a 50-person team is reviewing your code
4. **Compound knowledge** — every project should make the next one faster
